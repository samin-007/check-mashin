# چک‌ماشین — Context پروژه

## ایده اصلی
**رادار فرصت خرید + تشخیص کلاهبرداری خودرو**

لینک آگهی رو بده، ۳ ثانیه‌ای بهت می‌گیم: بخر، صبر کن، یا فرار کن.

---

## مخاطب هدف
- خریداران خودرو (هر کسی که فکر خرید ماشین داره)
- فروشنده‌ها (بفهمن ماشینشون رو چند بذارن)
- سرمایه‌گذاران خودرو (سیگنال خرید/فروش)
- نمایشگاه‌دارها (تحلیل بازار)

---

## قابلیت‌های اصلی

### ۱. چک آگهی (فیچر اصلی)
- کاربر لینک آگهی دیوار/باما رو می‌ده
- اپ مقایسه قیمت با بازار + امتیاز سلامت آگهی رو نشون می‌ده
- خروجی: فرصت خرید / قیمت منصفانه / بالای بازار / مشکوک

### ۲. رادار فرصت
- آگهی‌های زیر قیمت بازار رو پیدا می‌کنه
- فیلتر بر اساس مدل، شهر، بودجه
- Push notification وقتی فرصت مطابق شرایط کاربر اومد

### ۳. هشدار هوشمند
- کاربر شرایطش رو تنظیم می‌کنه (مثلاً "۲۰۷ زیر ۱.۲ میلیارد تهران")
- سیستم ۲۴ ساعته رصد می‌کنه و خبر می‌ده

### ۴. بازار
- شاخص کل بازار خودرو (مثل شاخص بورس)
- قیمت روز مدل‌ها
- مقایسه رشد خودرو vs دلار vs طلا vs بورس
- بیشترین رشد / ریزش روزانه

### ۵. تشخیص کلاهبرداری
- بررسی اصالت عکس‌ها
- چک تعداد آگهی‌های فروشنده (دلال vs شخصی)
- تحلیل متن آگهی
- گزارش‌های مردمی

## تصمیمات فنی

| بخش | تکنولوژی |
|------|----------|
| اپ موبایل | React Native (Expo SDK 54) |
| بک‌اند | Python (FastAPI) |
| دیتابیس | PostgreSQL |
| نوتیفیکیشن | Firebase Cloud Messaging |

### ساختار اپ:
- **تم:** تیره (Dark) — مثل داشبورد ماشین
- **رنگ اصلی:** سبز نئون `#00E676`
- **رنگ هشدار:** قرمز `#FF5252`
- **Navigation:** Bottom Tab — ۵ تب (خانه، چک آگهی، فرصت‌ها، هشدارها، بازار)

## نکات مهم برای توسعه‌دهنده
- پروژه Expo هست — برای اجرا: `npm install` → `npx expo start`
- تم تیره خودرویی — رنگ‌ها تو `src/theme/colors.js`
- داده‌ها فعلاً Mock هستن — بعداً به API وصل می‌شن
- کامپوننت‌ها reusable هستن — تو `src/components/`
- فارسی‌محور — RTL و تقویم شمسی لازمه (بعداً)
